there are the 5 types of request 
1. GET -- Retrieves information from the specific resource,  and should only be used to request data(not to modify it ) 
2. POST -- Sends data to the server for processing , usually resulting in a change in the server state or side effects on the server 
3. PUT -- updates a current resource or creates it if it does'nt exist, with the client providing a complete and updated copy of the resource -- the bicycle from amazon  example 
4. PATCH -- updates parts of an existing resource , with the client providing only the parts of the resource that need to be updated. -- the bicycle's tyre 's example 
5. DELETE -- removes the specific resource from the server
